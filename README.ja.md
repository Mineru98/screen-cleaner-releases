# ScreenCleaner

## このアプリを作った理由

正直に言うと、MacBookの画面って汚くなりますよね。

よし掃除しようとクロスを手に取ったら、手がキーボードをかすめてタブが吹き飛んで、ファイル名が「あああああ」になって、なぜかYouTubeが突然再生されて……

結果、画面を拭く前に5分間の後片付け。

画面はまだ汚いまま。

macOSにクリーニングモードがないから、作りました。

---

## 何をするアプリですか

**ScreenCleaner**はたった一つの役割に特化した小さなmacOSメニューバーアプリです：何も起こらないように安全に画面を拭ける時間を作ります。

**クリーニング開始**を押すと：

- **すべてのキーボード入力をシステムレベルでブロック** — どれだけキーボードを強く押しても、何も入力されません
- **画面を暗く黒くして**、汚れや筋が見えやすくなります
- **タイマーのカウントダウン**（1、3、5、10分またはカスタム）でクリーニング時間を管理します

タイマーが終わると、すべてがもとに戻ります。きれいな画面、混乱なし。

## インストール方法

### Homebrew（推奨）

```bash
brew tap Mineru98/tap
brew install --cask screen-cleaner
```

### 手動インストール

1. [Releases](https://github.com/Mineru98/screen-cleaner-releases/releases)から最新の`.dmg`をダウンロード
2. `.dmg`を開いて**ScreenCleaner.app**を`/Applications`にドラッグ
3. アプリを起動し、案内に従って**アクセシビリティ**の権限を許可

## 必要条件

| | |
|---|---|
| **macOS** | 14 Sonoma 以降 |
| **権限** | アクセシビリティ（キーボードブロックに必要） |
| **アーキテクチャ** | Apple Silicon & Intel |

---

## 使い方

1. アプリケーションまたはメニューバーから**ScreenCleaner**を起動
2. 初回起動時に**アクセシビリティ**の権限を許可
3. タイマープリセットを選択 — **1 / 3 / 5 / 10分** — またはカスタム時間を入力
4. **クリーニング開始**をクリック
5. 自由に画面を拭く
6. タイマーが終わると自動的にすべてが復元されます

### クリーニング中に使えるキー

クリーニングモード中は、以下の3つのキーだけが有効です。それ以外はすべてブロックされます。

| キー | 動作 |
|------|------|
| `F1` | 画面の輝度を下げる |
| `F2` | 画面の輝度を上げる |
| `ESC`（5秒長押し） | クリーニングをキャンセルして元に戻す |

> ESCを一度タップしても何も起きません。5秒間押し続けると終了します。

---

## 機能

- `CGEventTap`によるシステムレベルのキーボードブロック — いかなる入力も通過しません
- クリーニング中に汚れが見えやすいよう、黒いオーバーレイで画面を暗くします
- クラッシュセーフ：アプリが予期せず終了しても、輝度が自動的に復元されます
- ミニマルなUI — タイマーとボタンだけのシンプルな画面
- メニューバーに常駐し、必要なときだけ使えます
